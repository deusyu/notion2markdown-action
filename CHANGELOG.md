# 更新日志

All notable changes to this project will be documented in this file.

## [0.7.5] - 2025-06-08

### 🎬 新增功能
- ✅ 支持YouTube视频嵌入
- ✅ 支持Bilibili视频嵌入  
- ✅ 支持QQ视频嵌入
- ✅ 支持普通视频文件(mp4, mov, avi, wmv, flv, mkv)
- ✅ 支持Notion内嵌视频文件迁移到自定义对象存储

### 🔧 技术改进
- 新增视频块处理逻辑
- 新增视频URL转换器
- 支持多种视频平台的embed代码生成
- 集成视频文件迁移功能
- 兼容现有的图床配置，支持AWS S3/Cloudflare R2等对象存储

### 📚 文档更新
- 更新项目署名和作者信息
- 重构README结构符合现代软件工程规范
- 更新所有示例中的action引用
- 完善LICENSE版权信息
